# SPF Portal Builder — Mining Safety Platform

Low-code dashboard builder for mine safety data. Non-technical miners can build portals, dashboards, and apps using safety data served by SPF (Safety Platform of the Future).

## How to preview (GitHub Codespaces)

1. On this repo page, click the green **Code** button
2. Go to the **Codespaces** tab
3. Click **Create codespace on main**
4. Wait ~60 seconds. The app will start automatically.
5. When prompted, click **Open in Browser** to see the portal.

## Features

- Dashboard designer with drag-and-drop widgets
- Pre-built templates (Shift Handover, Incident Tracker, Compliance, Training, Equipment)
- SPF connector for live safety data from Snowflake
- Data Explorer to browse available data sources
- Configurable KPI cards, incident feeds, compliance gauges, alert panels
